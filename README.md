# Sistema de Academia em Java com Programação Orientada a Objetos

Este projeto é uma aplicação desenvolvida em Java, com foco na aplicação prática dos conceitos de Programação Orientada a Objetos (POO).
O sistema demonstra o uso de classes, encapsulamento, herança, métodos e polimorfismo, sendo executado em ambiente console.


## ⚙️ Funcionalidades

* Instanciação de objetos

* Uso de herança para reaproveitamento de código

* Aplicação de regras específicas conforme o tipo do objeto

* Exibição de resultados no console

* Estrutura clara e didática para fins educacionais


## Tecnologias Utilizadas

* Java
